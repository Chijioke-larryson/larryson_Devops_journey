# Docker DevOps Course Notes

This repository will be used to follow along with the freeCodeCamp course **"Learn Docker – Full DevOps Course for Deploying Containerized Apps"** by freeCodeCamp.org.

## Course Focus

The course is a hands-on introduction to Docker and containerization for DevOps workflows. It starts from the fundamentals and moves toward practical container-based application deployment.

What we will cover:

- What containers are and why Docker matters
- The difference between physical servers, virtual machines, and containers
- Docker architecture and core concepts
- Installing and using Docker in a cloud environment
- Connecting to an EC2 instance with Session Manager and SSH
- Building and running containerized applications
- Working with Docker images and containers
- Understanding the basics needed to deploy apps reliably in containers
